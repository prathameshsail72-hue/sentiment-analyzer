# Smart Sentiment Analyzer

A modern web application built with Flask and Hugging Face's distilbert-base-uncased model for natural language processing. 

## Features
- Deep learning sentiment analysis
- Dynamic REST API endpoint `/analyze`
- Responsive and modern UI styled with Bootstrap 5
- Prepared for deployment on Google Cloud Run

## Installation (Local)
1. Clone the repository and install requirements:
   ```bash
   pip install -r requirements.txt
   ```
